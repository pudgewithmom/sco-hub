# SCO Youth Talent Hub — Demo Prototype

Демонстрационный прототип платформы для проектных команд и карьерных возможностей ШОС.

## Деплой на GitHub Pages

1. Создай репозиторий на GitHub (например `sco-youth-hub`)
2. Загрузи все файлы из этой папки в корень репозитория
3. Перейди в Settings → Pages → Source: `main` branch, `/ (root)`
4. Сайт будет доступен по адресу: `https://<username>.github.io/sco-youth-hub/`

## Структура

```
index.html          — Главная страница
demo.html           — Страница для QR-кода с конференции
projects.html       — Каталог проектов
talents.html        — Каталог специалистов
apply.html          — Форма отклика на проект
success.html        — Страница подтверждения
about.html          — О платформе
pilot.html          — Форма участия в пилоте

projects/           — Страницы отдельных проектов
talents/            — Страницы отдельных специалистов
data/               — Статические данные (JS-объекты)
js/                 — Логика страниц
css/                — Стили
```
